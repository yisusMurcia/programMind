En este espacio se presentan diferetnes códigos que he ehecho para solucionar problemas haceindo uso de herramientas de inteligenica artificial.

Tree.py

Este archivo es usado en varios de los algoritmos, en este se crea una clase con diferentes propiedades, tales como tener un nodo padre e hijos o poseer un coste e información. De esta manera, los nodos creados actuan como un digrama de arbol

tree.py

Problemas

Puzzle

Este problema consiste en organizar una lista de números de orden ascendente, para ello se va intercambiando de lugar las piezas del puzzle, el algoritmo busca armar el puzzle en la menor cantidad de pasos.

Viaje de carretera

Este problema conciste en una conexión entre ciudades, donde se tiene una ciudad de partida y una de llegada, por lo que se debe buscar el camino con mejor distancia entre las dos ciudades.

Camino de grafos

Este algoritmo busca unir una serie de grafos, representados como diferentes rutas con un coste cada una, de tal manera que todos los grafos sean recorridos.

Inecuación

El código busca resolver una inecuación con dos incognitas, la idea es buscar el mayor valor posible para las variables.

7x + 4y ≤ 150
6x + 5y ≤ 160

Empresa de viaje

Dada una cantidad de pedidos a diferentes ciudades, se queiere crear rutas de camiones que optimicen la entrega de los pedidos, sin exceder la carga máxima.

ALgoritmo de costos

Se requieren 4 productos de 4 empresas que los ofrecen con diferentes costos, pero se quiere que cada empresa aporte uno de esos productos a la vez que se toma el menor costo posible.

Puzzle.py

Este código busca organizar un rompecabezas de números (va de 1 a 4), requiere de conocer la solución del rompecabezas y el orden inical de los números, la salida del código es la secuencia de pasos para resolver el rompecabezas.

RoadTrip.py

El código busca crear la mejor ruta para ir de una ciudad a otra, para ello requiere las conexiones entre las carreteras (rutas) y se emplean las coordenadas para hacer un estimación de la distancia, el código retorna el mejor camino para ir de una ciudad a otra.

RoadTrip.py

GraphTraversal.py

Él codigo busca recorrer un camino de puntos, estos puntos conectan con otros puntos a determinada distancia, por lo que se busca es recorrer todos los puntos con el menor coste posible, el código retorna dicho camino.

GraphTraversal.py

Inequation.py

El código busca resolver una inecuación con dos incognitas, la idea es buscar el mayor valor posible para las variables.

7x + 4y ≤ 150
6x + 5y ≤ 160

inequation.py

SavingsAlgorithm

Este código busca crear rutas para entregar una serie de paquetes, la ruta debe ser la mejor, para ello se tiene en cuanta aquellos pedidos que se pueden conectar en una misma ruta y que no sobrepasen el peso máximo.

savingsAlgorithm.py

SearchMinCost

El código busca encontrar la mejor forma de organizar un pedido donde se tomen cuatro tipos y se compren uno por cada proveedor y aquel pedido tenga el menor coste.

searchMinCost.py
