Todos deseamos tener tiempo libre, hay tantas cosas que hacer que nos quitan ese tiempo libre, por esos siempre que nos quedan ese tiempo libre buscamos disfrutarlo.

Sin embargo, cuando tenemos este tiempo libre lo desperdiciamos, nos quedamos pegados a una pantalla donde terminamos perdiendo el tiempo, y aunque este tiempo se puede usar para descansar, usandolo de esta manera no nos ayuda, tras una pantalla no descansamos, nuestro cerebro se sigue cansando

¿Qué hacer en el tiempo libre

Se puede emplear el tiempo libre para tomar un descanso de verdad, alejado de todo trabajo y dispositivos, o de una forma productiva que de valor a nuestras vidas, de esta manera encontramos dos formas: Disfrutar del ahora Mejorar nuestras vidas

Disfrutar del ahora

Una de las mejores maneras para usar de nuestro tiempo libre, es emplandolo para disfrutar, per o disfrutar del presente, compartiendo con amigos o familia, o teninedo un tiempo para nosotros mismos haceindo eso que nos gusta.

Mejorar nuestra vida

Con esta opción se busca aprovechar nuestro tiempo para construir un vida mejor, ya sea aprendiendo algo nuevo que nos sirva en el ambito laboral o para nuestras vidas, para empezar a emprender, o también para mejorar cualquier otro aspecto de nuestra vida.
